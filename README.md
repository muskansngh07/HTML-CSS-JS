# ☆ About ☆

30 days of pure HTML- no CSS, no Javascript.

## Day 01

Learnt the following:

- `<!doctype html>` is used to ensure proper rendering of the code as html, without any abnormalities.
- `<head>` contains the content not to included when a viewer loads the web page.
- `<meta charset="utf-8" />` covers the range of all possible letters and symbols.
- `<meta name="viewport" content="width=device-width" />` ensures that the web page loads to the full width of the device it is being accessed on. 
- `<title>` tells the name of the tab in which the page is contained.
- `<body>` contains the content displayed on the web page for the user.
- `<h1>` denotes the primary heading.
- `<p>` denotes a paragraph.
- `<li>` denotes a list.
