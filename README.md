## All the things I learnt in HTML

<h3 align="center">Tags and their Use</h3>

<table align="center">
  <tr>
    <th>S No.</th>
    <th>Tags</th>
    <th>Name</th>
    <th>Use</th>
  </tr>
  <tr>
    <th>1. </th>
    <td> &lt;!DOCTYPE html&gt; </td>
    <td>Document Type Declaration</td>
    <td>It’s an instruction that tells the browser to render the page in "Standards Mode" using HTML5.</td>
  </tr>
  <tr>
    <th>2. </th>
    <td>&lt;html&gt;</td>
    <td>The Root element</td>
    <td>The ultimate parent that wraps all other content. It defines the boundaries of the HTML document and often includes the language attribute (e.g., `<html lang="en">`).</td>
  </tr>
  <tr>
    <th>3. </th>
    <td>&lt;head&gt;</td>
    <td>The Document Metadata Container</td>
    <td>It acts as the "brain" of the page, holding machine-readable information that isn't shown to the user, such as character encoding, links to CSS stylesheets, and scripts.</td>
  </tr>
  <tr>
    <th>4. </th>
    <td>&lt;body&gt;</td>
    <td>The Body</td>
    <td>Contains all the data visible on the web page</td>
  </tr>
  <tr>
    <th>5. </th>
    <td>&lt;h1&gt; to &lt;h6&gt;</td>
    <td>Section Headings</td>
    <td>A six-level hierarchy used to outline the structure of your content. &lt;h1&gt; is the most important while &lt;h6&gt; is the least.</td>
  </tr>
  <tr>
    <th>6. </th>
    <td>&lt;p&gt;</td>
    <td>Paragraph</td>
    <td>Used to group together sentences of related text.</td>
  </tr>
  <tr>
    <th>7. </th>
    <td>&lt;a&gt;</td>
    <td>Anchor (Hyperlink)</td>
    <td>Used to link to other pages or files. It uses the href attribute to define the destination.</td>
  </tr>
  <tr>
    <th>8. </th>
    <td>&lt;img&gt;</td>
    <td>Image</td>
    <td>An "empty" or <strong>self-closing</strong> tag used to embed pictures. It requires a src (source) for the file path and alt (alternative text) for accessibility.</td>
  </tr>
  <tr>
    <th>9. </th>
    <td>&lt;ul&gt;</td>
    <td>Unordered List</td>
    <td>Creates bullet points for text.</td>
  </tr>
  <tr>
    <th>10. </th>
    <td>&lt;ol&gt;</td>
    <td>Ordered List</td>
    <td>Creates numbered list for text. </td>
  </tr>
  <tr>
    <th>11. </th>
    <td>&lt;li&gt;</td>
    <td>List Items</td>
    <td>Mandatory child tag for both &lt;ul&gt; and &lt;ol&gt;</td>
  </tr>
  <tr>
    <th>12. </th>
    <td>&lt;div&gt;</td>
    <td>Division/Container</td>
    <td>A generic box used to group elements together for layout purposes or styling with CSS. It has no visual effect until you add styles.</td>
  </tr>
  <tr>
    <th>13. </th>
    <td>&lt;strong&gt;</td>
    <td>Bold</td>
    <td>Makes text bold</td>
  </tr>
  <tr>
    <th>14. </th>
    <td>&lt;em&gt;</td>
    <td>Italic</td>
    <td>Makes text italic</td>
  </tr> 
  <tr>
    <th>15. </th>
    <td>&lt;hr&gt;</td>
    <td>The Horizontal Rule</td>
    <td>A <strong>self-closing</strong> that represents a thematic break between paragraph-level elements.</td>
  </tr>  
  <tr>
    <th>16. </th>
    <td>&lt;table&gt;</td>
    <td>The Table Wrapper</td>
    <td>Used to arrange data in the form of rows and columns.</td>
  </tr>
  <tr>
    <th>17. </th>
    <td>&lt;tr&gt;</td>
    <td>Table Row</td>
    <td>Defines a single horizontal row of cells.</td>
  </tr>
  <tr>
    <th>18. </th>
    <td>&lt;th&gt;</td>
    <td>Table Header</td>
    <td>Used for the first row or column to define the labels. Text inside is usually bold and centered by default.</td>
  </tr>
  <tr>
    <th>19. </th>
    <td>&lt;td&gt;</td>
    <td>Table Data</td>
    <td>The standard cell that holds the actual content (text, images, or links).</td>
  </tr>
  <tr>
    <th>20. </th>
    <td>&lt;blockquote&gt;</td>
    <td>The Block Quotation</td>
    <td>Used for long, multi-line quotes. It is a "block-level" element, meaning it starts on a new line and takes up the full width available.</td>
  </tr>
  <tr>
    <th>21. </th>
    <td>&lt;section&gt;</td>
    <td>The Thematic Section</td>
    <td>Used to define a standalone portion of a document, where related elements are grouped. Helps browsers and search engines work better.</td>
  </tr>
  <tr>
    <th>22. </th>
    <td>&lt;iframe&gt;</td>
    <td>The Inline Frame</td>
    <td>An element that allows you to nest an entirely different webpage within your own. Commonly used for embedding third-party content like YouTube videos, Google Maps, or social media posts.</td>
  </tr>
  </table>
