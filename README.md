## All the things I learnt in HTML

<h3 align="center">Tags and their Use</h3>

<table>
  <tr>
    <th>Tags</th>
    <td>Name</td>
    <td>Use</td>
  </tr>
  <tr>
    <td> &lt;!DOCTYPE html&gt; </td>
    <td>Document Type Declaration</td>
    <td>It’s an instruction that tells the browser to render the page in "Standards Mode" using HTML5.</td>
  </tr>
  <tr>
    <td>&lt;html&gt;</td>
    <td>The Root element</td>
    <td>The ultimate parent that wraps all other content. It defines the boundaries of the HTML document and often includes the language attribute (e.g., `<html lang="en">`).</td>
  </tr>
  <tr>
    <td>&lt;head&gt;</td>
    <td>The Document Metadata Container</td>
    <td>It acts as the "brain" of the page, holding machine-readable information that isn't shown to the user, such as character encoding, links to CSS stylesheets, and scripts.</td>
  </tr>
  <tr>
    <td>&lt;body&gt;</td>
    <td>The Body</td>
    <td>Contains all the data visible on the web page</td>
  </tr>
  <tr>
    <td>&lt;h1&gt; to &lt;h6&gt;</td>
    <td>Section Headings</td>
    <td>A six-level hierarchy used to outline the structure of your content. &lt;h1&gt; is the most important while &lt;h6&gt; is the least.</td>
  </tr>
  <tr>
    <td>&lt;p&gt;</td>
    <td>Paragraph</td>
    <td>Used to group together sentences of related text.</td>
  </tr>
  <tr>
    <td>&lt;a&gt;</td>
    <td>Anchor (Hyperlink)</td>
    <td>Used to link to other pages or files. It uses the href attribute to define the destination.</td>
  </tr>
  <tr>
    <td>&lt;img&gt;</td>
    <td>Image</td>
    <td>An "empty" or <strong>self-closing</strong> tag used to embed pictures. It requires a src (source) for the file path and alt (alternative text) for accessibility.</td>
  </tr>
  <tr>
    <td>&lt;ul&gt;</td>
    <td>Unordered List</td>
    <td>Creates bullet points for text.</td>
  </tr>
  <tr>
    <td>&lt;ol&gt;</td>
    <td>Ordered List</td>
    <td>Creates numbered list for text. </td>
  </tr>
  <tr>
    <td>&lt;li&gt;</td>
    <td>List Items</td>
    <td>Mandatory child tag for both &lt;ul&gt; and &lt;ol&gt;</td>
  </tr>
  <tr>
    <td>&lt;div&gt;</td>
    <td>Division/Container</td>
    <td>A generic box used to group elements together for layout purposes or styling with CSS. It has no visual effect until you add styles.</td>
  </tr>
  <tr>
    <td>&lt;strong&gt;</td>
    <td>Bold</td>
    <td>Makes text bold</td>
  </tr>
  <tr>
    <td>&lt;em&gt;</td>
    <td>Italic</td>
    <td>Makes text italic</td>
  </tr>  
  </table>
